## How to run application

1. Open code editor and clone repository
 ```
 git clone https://github.com/Efren707/hedgeServe.git
 ```
2. Install dependencies
```
npm install
```

5. Run server
```
ng serve --open
```

7. Navigate to your browser and go to http://localhost:4200/
